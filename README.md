# zk-project

Simulación inicial de un sistema Zero Knowledge.

## Objetivo
Demostrar que un score privado cumple una condición pública sin mostrar el valor exacto.

## Ejemplo
- Score privado: 812
- Threshold público: 750
- Resultado esperado: true

## Estructura
- `index.js`: ejecuta el flujo principal
- `prover.js`: genera la prueba
- `verifier.js`: verifica la prueba
